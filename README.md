Quiz App
This is a simple Quiz App built using HTML, CSS, and JavaScript. It presents users with multiple-choice questions, tracks their scores, and displays the results at the end.

Features:
1. Multiple Categories
2. Responsive Design
3. Dynamic quiz

Tecghnologies Used
1.HTML
2.CSS
3. Javascript
